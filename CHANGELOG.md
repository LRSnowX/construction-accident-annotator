# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-11-14

### 新增

- 🎯 交互式命令行标注界面
- 💾 断点续传功能，支持随时暂停和继续
- ↶ 撤销功能，支持撤销上一个标注
- ⊘ 跳过功能，允许暂时跳过不确定的案例
- 📊 实时显示标注进度和统计信息
- 🔄 自动保存功能，每10个案例自动保存
- 🎲 随机标注模式，适用于多人协作场景
- 👥 多人协作支持，不同标注者独立文件
- 📁 智能文件检测，自动识别并选择CSV文件
- 📈 双格式输出：CSV（人工审查）+ Parquet（快速加载）
- 🖥️ 双击启动脚本（macOS/Windows）
- 🔀 标注结果合并工具（merge_annotations.py）
- 📚 完整的使用文档和快速入门指南
- 📖 建筑业分类国家标准参考（GB-T4757-2017）

### 功能特性

- 支持 Python 3.8+
- 跨平台支持（macOS、Windows、Linux）
- 零代码上手，适合非程序员使用
- 自动环境检查和依赖安装
- 冲突检测和统计分析
