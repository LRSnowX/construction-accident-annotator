# 更新日志

记录项目的重要更改。

## 2024-11-15

### 新增
- 🔄 用户名记忆功能：自动保存上次使用的用户名，下次启动直接回车即可使用
- ⏭️ 自动跳过已标注案例：支持外部编辑（如Excel）后继续标注，避免重复劳动
- 📊 精确的进度统计：显示真实已标注数量、建筑业/非建筑业分类统计

### 修复
- 🐛 修复索引越界错误：跳过已标注案例时的边界处理问题
- 🐛 修复撤销功能bug：现在能正确回退到上一个标注案例（支持顺序/随机模式）
- 🐛 修复自动保存逻辑：基于实际标注数量而非进度位置触发保存

### 优化
- 🎨 改进随机模式下的进度提示，更清晰易懂
- 📝 CSV格式要求说明更通用化：只需 `full_text` 列，其他列自动识别
- 🔧 display_case 函数自动适配数据格式：只显示实际存在的字段
- 📝 更新.gitignore：添加用户名记录文件、macOS和Python版本管理文件

## 2024-11-14

### 初始版本

### 新增

- 🎯 交互式命令行标注界面
- 💾 断点续传功能，支持随时暂停和继续
- ↶ 撤销功能，支持撤销上一个标注
- ⊘ 跳过功能，允许暂时跳过不确定的案例
- 📊 实时显示标注进度和统计信息
- 🔄 自动保存功能，每10个案例自动保存
- 🎲 随机标注模式，适用于多人协作场景
- 👥 多人协作支持，不同标注者独立文件
- 📁 智能文件检测，自动识别并选择CSV文件
- 📈 双格式输出：CSV（人工审查）+ Parquet（快速加载）
- 🖥️ 双击启动脚本（macOS/Windows）
- 🔀 标注结果合并工具（merge_annotations.py）
- 📚 完整的使用文档和快速入门指南
- 📖 建筑业分类国家标准参考（GB-T4757-2017）

### 功能特性

- 支持 Python 3.8+
- 跨平台支持（macOS、Windows、Linux）
- 零代码上手，适合非程序员使用
- 自动环境检查和依赖安装
- 冲突检测和统计分析
