# 启动说明

## 🖱️ 如何启动标注工具

### macOS 用户

双击这个文件启动：
```
启动标注工具.command
```

**首次使用可能需要额外步骤**：
1. 右键点击 `启动标注工具.command`
2. 按住 `Option` 键，选择"打开"
3. 在弹出的对话框中点击"打开"
4. 之后就可以直接双击启动了

### Windows 用户

双击这个文件启动：
```
启动标注工具.bat
```

**如果出现安全警告**：
1. 点击"更多信息"
2. 点击"仍要运行"
