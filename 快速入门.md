# 快速入门指南（给非程序员）

## 📋 准备工作

### 第一步：获取项目文件

1. 将整个项目文件夹复制到你的电脑上
2. 确保文件夹名称为 `construction-accident-annotator`

### 第二步：准备数据文件

1. 找到项目文件夹中的 `data` 文件夹
2. 打开 `data` 里的 `raw` 文件夹
3. 将你的CSV数据文件放进去（文件名可以任意）

**重要**：CSV文件必须包含 `full_text` 列（存放事故案例文本），其他列为可选

完成后的文件结构应该是这样：
```
construction-accident-annotator/
├── data/
│   └── raw/
│       └── 你的数据.csv  ← CSV文件放这里（任意名称）
├── 启动标注工具.command   ← macOS用户点这个
└── 启动标注工具.bat       ← Windows用户点这个
```

## 🚀 开始标注

### macOS 用户

1. 找到 `启动标注工具.command` 文件
2. 双击运行
3. 如果提示"无法打开"：
   - 右键点击文件
   - 选择"打开方式" → "终端"
   - 或者：系统偏好设置 → 安全性与隐私 → 点击"仍要打开"

### Windows 用户

1. 找到 `启动标注工具.bat` 文件
2. 双击运行
3. 如果出现安全警告，点击"更多信息" → "仍要运行"

## 💡 使用说明

### 1. 启动后会看到

```
==================================
  建筑业事故案例标注工具
==================================

正在启动程序...
✓ 检测到虚拟环境
✓ 检查依赖包...
✓ 所有检查通过

==================================

================================================================================
                     建筑业事故案例标注系统
================================================================================
```

**如果有多个CSV文件**，会看到文件选择界面：
```
📂 检测到 3 个CSV文件，请选择要标注的文件：

  1. accident_cases_2024.csv (263.0 MB)
  2. accident_cases_2023.csv (180.5 MB)
  3. test_data.csv (5.2 MB)

请输入文件序号: 
```
输入序号（如 `1`）选择文件。

### 2. 输入你的名字

例如输入：`张三`（或任何你想用的名字）

### 3. 选择是否随机模式

```
🎲 是否启用随机标注模式？（多人协作时建议启用，避免冲突）
   请选择 (y/n, 默认n): 
```

- 如果只有你一个人标注：输入 `n` 或直接回车
- 如果有多人同时标注：输入 `y`

### 4. 开始标注

程序会显示案例内容，你需要判断：

```
请标注此案例:
  1 - 建筑业案例
  0 - 非建筑业案例
  s - 跳过此案例
  u - 撤销上一个标注
  q - 保存并退出

请输入: 
```

**标注指南**：
- 按 `1`：如果是建筑业相关的事故
- 按 `0`：如果不是建筑业相关的事故
- 按 `s`：如果不确定，先跳过
- 按 `u`：撤销刚才的标注
- 按 `q`：保存并退出（可以随时退出，下次继续）

**💡 不清楚什么是建筑业？**

如果不确定某个案例是否属于建筑业，可以参考项目中的 `GB-T4757-2017.md` 文件，里面详细列出了：
- 房屋建筑业（住宅、体育场馆等）
- 土木工程建筑业（道路、桥梁、管道、电力等）
- 建筑安装业（电气、管道安装等）
- 建筑装饰装修业

简单来说，凡是涉及建筑施工、安装、装修的事故都算建筑业。

## ❓ 常见问题

### 程序无法启动？

**macOS**：
- 右键点击 `启动标注工具.command`
- 选择"打开方式" → "终端"

**Windows**：
- 确保已安装Python（可以问IT部门）
- 右键以管理员身份运行

### 找不到数据文件？

确保CSV文件放在了正确的位置：
```
项目文件夹/data/raw/你的文件.csv
```

**注意**：CSV文件必须包含 `full_text` 列，其他列（如 title、url 等）为可选，程序会自动识别。

### 想暂停标注？

直接按 `q` 退出，下次打开会从上次位置继续。

### 不小心标错了？

按 `u` 可以撤销上一个标注。

### 多人同时标注？

- 每个人使用不同的名字（用户名）
- 都选择"随机模式"（输入 `y`）
- 这样可以避免标注同样的案例
